# Express-сервер

### Описание проекта:
Это REST API сервер на Node.js через веб-фреймворк express.js, база данных - MongoDB, интерфейс Mongoose.

### Как запускать проект:
`npm run start` для запуска сервера на http://localhost:3000,  
`npm run dev` откроет в браузере с hot-reload (установите nodemon)
